# Reviewer queue share quick note

Use this when you need one short reviewer handoff sentence from the markdown or PR-comment summary.

## Minimal recipe

1. Read `Reviewer queue next-focus priority label`.
2. Read `Reviewer queue next-focus queue share`.
3. Read `Reviewer queue next-focus source-case rate`.
4. Paste one sentence in the form:
   - `Start with <priority label>; it covers <queue share> of queued follow-up and <source-case rate> of source cases.`

## Example

- `Start with P1 · fix regressions; it covers 66.67% of queued follow-up and 40.00% of source cases.`

## Why this exists

It gives release reviewers a deterministic, copy-ready queue note without reopening the JSON artifact.
