# Reviewer queue report bundle handoff card

Use one sentence that keeps the reviewer queue decision and the saved artifact bundle together:

- Winner: name the queue or reviewer lane first.
- Why now: mention the narrow reason the queue is ahead (share, pass rate, source-case impact, or tie mode).
- Bundle: keep the shared JSON/Markdown/HTML artifact path in the same sentence.

Example:

`Lane B stays next because it leads on source-case impact; reopen artifacts/review-lane-b-summary.json, artifacts/review-lane-b-summary.md, and artifacts/review-lane-b-summary.html together.`
