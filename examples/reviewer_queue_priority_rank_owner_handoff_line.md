# Reviewer queue priority-rank owner handoff line

Use this when the priority rank is already exposed and you need a single owner-facing line.

## Pattern

`Next reviewer focus: <queue> (rank <n>); owner <owner> should clear <reason> before the runner-up <runner_up>.`

## Keep visible

- the exposed rank
- the owner or lane taking the next pass
- the reason the queue is still first
- the runner-up if the lead is narrow

## Hold rule

If the owner is unknown or the winner is only tied by rank/share, do not post this line alone; pair it with the tie or ready-signal guide first.
