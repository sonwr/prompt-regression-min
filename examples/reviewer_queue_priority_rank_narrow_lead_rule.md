# Reviewer Queue Priority Rank Narrow-Lead Rule

Use this when one reviewer queue clearly leads, but the margin is still small enough that you should not oversell certainty.

Rule:
- Name the current winner.
- Keep the runner-up visible if the lead is still narrow.
- Point reviewers to the shared artifact bundle, not just the headline rank.
- Avoid release-ready language unless the queue gap and proof assets both justify it.
