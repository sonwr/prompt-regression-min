# Reviewer queue advantage direction quickstart

Use this when `summary.reviewer_queue.next_focus_advantage_direction` is already available and you only need the shortest human handoff.

## Read the direction first

- `none` — no reviewer queue is active yet.
- `tied` — more than one queue is tied for the lead; keep the tie visible.
- `solo` — only one queue exists; route it directly.
- `ahead` — one queue is ahead of the runner-up; name both if the margin feels small.

## Small posting rule

- `none` → say there is no queued follow-up.
- `tied` → keep the tied first lanes visible in the reviewer note.
- `solo` → post one clear owner-ready next step.
- `ahead` → post the winner first, then keep the runner-up visible if useful.

## Minimal example lines

- `none`: No reviewer queue is active.
- `tied`: Next focus is tied between regressions and watchlist review.
- `solo`: Next focus is fix regressions.
- `ahead`: Next focus is fix regressions; watch unchanged fails stays second.
