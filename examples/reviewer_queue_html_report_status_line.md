# Reviewer queue HTML report status line

Use this when the HTML report is ready and you need one short reviewer-facing update.

## Status line pattern

`HTML report ready: <queue> remains the next focus (<share>% queue share, <cases> source cases), with <runner-up> still visible as the runner-up.`

## Quick checks

- Name the winning queue.
- Keep queue share visible.
- Keep source-case impact visible.
- Mention the runner-up when the lead is narrow.

## Example

`HTML report ready: policy remains the next focus (44% queue share, 11 source cases), with formatting still visible as the runner-up.`
