# Reviewer queue source-case rate note

Use this example when the active-case window is empty but filtered or skipped cases still matter in the broader dataset.

- Prefer `source_case_rate` when `active_cases=0` and you still need dataset-level follow-up weight.
- Keep `active-case rate` visible too, but do not pretend it explains the whole queue when there is no active window.
- A short handoff line can say: `confirm_filtered_scope still covers 40.00% of source cases even though the active window is empty.`
