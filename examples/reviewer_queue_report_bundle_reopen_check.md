# Reviewer queue report bundle reopen check

Use this when the JSON summary, HTML report, and queue handoff sentence all changed in the same slice.

## 30-second reopen loop

1. Open the emitted JSON summary and confirm the chosen reviewer queue still leads.
2. Open the matching HTML report and confirm the same queue label is visible there.
3. Re-read the handoff sentence and make sure it names the same queue and artifact bundle.
4. If any one of those three drifts, rerun the report bundle before posting the handoff.
