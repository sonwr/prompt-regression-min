# Reviewer queue priority-rank margin note

Use this example when the reviewer queue already has a winner, but you still need a short sentence that explains **why the lead is believable**.

Keep the handoff compact:
- name the winning queue key;
- mention the runner-up queue;
- cite the case-count gap and queue-share gap together.

This keeps follow-up priority posts grounded in reproducible queue math instead of vague urgency.
