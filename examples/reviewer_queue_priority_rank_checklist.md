# reviewer queue priority-rank checklist

Use this checklist when the summary already exposes reviewer queue priority ranks, but you want a fast pre-post audit before writing the handoff sentence.

1. Confirm the named next-focus queue is also the best priority rank among active queues.
2. If two queues share the same size, keep the lower rank number first.
3. If a lower-priority queue is larger, say that explicitly in the handoff instead of hiding the tradeoff.
4. Mention runner-up context when the lead is narrow.
5. Keep the final sentence deterministic: queue label, rank, case count, and why it is first.

## Suggested sentence frame

`Next focus: <label> (P<rank>, <cases> cases) because it stays highest in the deterministic follow-up order.`
