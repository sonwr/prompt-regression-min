# Reviewer queue artifact reopen line

Use this when the queue recommendation is already known and reviewers mainly need to reopen the right artifact bundle.

Template:

`Next reopen: <queue label> queue first — start with <summary artifact path>, then check <bundle artifact path> before broader triage.`

Example:

`Next reopen: policy-review queue first — start with artifacts/summary.md, then check artifacts/reviewer-queue.html before broader triage.`
