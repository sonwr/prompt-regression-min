# Reviewer queue report bundle priority note

Use this note when a short reviewer handoff needs to keep the selected queue tied to one saved artifact bundle.

## Rule

Name the winning reviewer queue first, then name the shared JSON/Markdown/HTML bundle, then name the owner who should reopen it.

## Why

That ordering keeps the handoff actionable without forcing reviewers to infer which artifact bundle goes with the current queue winner.

## One-line template

`Queue <label> stays first; reopen <basename>.json/.md/.html, then hand off to <owner>.`
