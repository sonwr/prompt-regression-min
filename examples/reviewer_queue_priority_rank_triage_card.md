# reviewer queue priority-rank triage card

Use this compact card when a regression report is already available and you only need the next review lane.

1. Read `reviewer_queue.follow_up_priority_summary`.
2. Start with `reviewer_queue.next_focus_handoff_summary`.
3. If tied, mention `reviewer_queue.next_focus_tie_summary`.
4. If needed, keep the runner-up visible with `reviewer_queue.runner_up_handoff_summary`.
