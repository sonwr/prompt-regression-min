# Reviewer queue next-focus owner note

Use this note when the summary already identifies the next reviewer focus and you want a short owner-facing update.

## Template

```text
Next reviewer focus: <queue label> (rank <n>, <share>% of active cases). <owner or team> should take the next pass because it currently leads the review queue.
```

## Example

```text
Next reviewer focus: Policy Review (rank 1, 42.9% of active cases). Safety owner should take the next pass because it currently leads the review queue.
```

## Quick check

Keep the note to one sentence and include the queue label, exposed rank, queue share, and owner/team.
