# Reviewer queue source-case scope note

Use this when the active shard is small but the source-case impact still matters.

Recommended one-line handoff shape:

- next focus queue,
- active-case rate,
- source-case rate,
- whether the runner-up still needs explicit review.

Keep the note honest about scope: a queue can dominate the current shard while still touching only a small share of the full source dataset.
