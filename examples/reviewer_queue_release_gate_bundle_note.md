# Reviewer Queue Release Gate Bundle Note

Use this note when the priority-rank winner is already known and the release gate handoff should also keep the shared JSON/Markdown/HTML artifact bundle visible.

## Short handoff

Post the release gate with the winning queue, the runner-up if the margin is still narrow, and the shared report bundle path in the same sentence.

## Why it exists

This keeps release-gate wording aligned with the exact artifact bundle reviewers should reopen before approving the next ship/no-ship call.
