# Reviewer queue bundle + queue mix note

Use this note when you want a short handoff that keeps the current report bundle path and queue mix summary in one reviewable block.

Example:
- bundle: `artifacts/summary.md`, `artifacts/summary.pr-comment.md`
- queue mix: `fix_regressions=3 case(s)/60.00% queue share/30.00% active-case rate/15.00% source-case rate`
- handoff: validate the bundle first, then post the queue mix line without widening the scope
