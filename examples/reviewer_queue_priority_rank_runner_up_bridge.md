# Reviewer Queue Priority Rank Runner-up Bridge

Use this when the priority-rank winner is already clear enough to act on, but you still want the runner-up queue to stay visible in one sentence.

## Bridge rule

- Name the winner first.
- Keep the runner-up visible without turning it into a tie.
- End with one concrete next action.

## Template

`Priority-rank winner: <winner> stays first; keep <runner-up> visible as the runner-up while <next action> closes the current lead.`

## Example

`Priority-rank winner: policy stays first; keep billing visible as the runner-up while the markdown/html summary bundle closes the current lead.`
