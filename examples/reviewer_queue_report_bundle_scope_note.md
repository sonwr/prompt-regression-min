# Reviewer queue report bundle scope note

Use this note when a reviewer handoff needs to mention the queue winner and the shared JSON/Markdown/HTML bundle without implying a full rerun.

## One-line rule

Name the active queue first, then point to the shared report bundle, and keep the handoff scoped to that queue.

## Example

`Next reviewer focus: fix regressions first; reopen the shared report bundle (JSON/Markdown/HTML) for the same shard before widening scope.`

## Why

This keeps queue triage, artifact naming, and report-bundle reopening aligned in one deterministic sentence.
