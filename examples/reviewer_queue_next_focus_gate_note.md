# Reviewer queue next-focus gate note

Use this note when a rerun is green enough to share but reviewers still need one clear next-focus queue plus one saved artifact bundle.

## Keep the handoff compact

- Name the dominant reviewer queue.
- Keep the saved summary/report artifact path in the same line.
- Avoid mixing queue priority, blocker explanation, and release narrative into separate fragments.

## Example shape

`next focus: fix_regressions -> artifacts/shard-a/summary.json`

That keeps queue ownership and artifact reopening in one replayable sentence.
