# Reviewer queue post-ready check

Use this note when the summary is almost ready and you only need a final yes/no before posting the reviewer handoff.

## Fast check

1. Confirm the winning queue label is still present in the summary JSON.
2. Confirm the runner-up or tie note is included when the queue lead is narrow.
3. Confirm the PR comment sentence matches the selected queue name and share.

## Exit phrases

- `post now` — queue winner is clear and the handoff sentence matches the payload.
- `add tie note` — top queues are too close; include the tie context before posting.
- `recompute queue summary` — metadata and handoff sentence disagree.
