# Reviewer Queue Priority Rank Quick Post

Use this when the priority-rank winner is already exposed and you need one short reviewer-facing update.

## Quick post rule

- Name the winning queue first.
- Mention whether the lead is clear or narrow.
- Keep the next action visible in the same sentence.

## Template

`Priority-rank winner: <queue> remains the next reviewer focus (<lead cue>); review <next action> before expanding to the runner-up.`

## Example

`Priority-rank winner: policy remains the next reviewer focus (clear lead); review the HTML-ready summary bundle before expanding to the billing runner-up.`
