# CLI Reviewer Queue Owner Lane Scope

Use this note when the reviewer queue summary already identifies the hottest lane and you only need a clean handoff sentence.

Keep the first handoff to three things:
1. the owner lane to focus first,
2. the queue metric that proves why it is hottest,
3. the smallest rerun or review cue that keeps the next check reproducible.

Avoid broad backlog storytelling when one owner lane already explains the immediate follow-up.
